About

The aim of the project is to improve on spellchecking for Nguni Languages, with emphasis on isiZulu and isiXhosa by: Providing the African Language Section at the University of Cape Town with an isiXhosa error detector. Providing the Language Department at the University of Kwa-Zulu Natal with an isiZulu error corrector. To compare which approach is more accurate at error detection for isiXhosa: statistical or rule-based approach.

Conclusion

Both the statistical and rule-based approach error detectors have been successfully implemented into standalone applications. The rule based approach is more accurate than the statistical based approach for the error detection. The error corrector has been successfully implemented and integrated with the already existing isiZulu error detector.

Rule-based spelling error detector for isiXhosa implemented using finite state transducers, regular expressions, SFST and Java Swing.

IsiZulu error corrector using a statistical approach integrated onto existing error detector. Trigrams, minimum-edit distance and probabilities were used to find candidate corrections.

IsiXhosa spellchecker created using data-driven statistical model. Character trigrams and probabilities were used to determine whether a word is correctly spelled.
